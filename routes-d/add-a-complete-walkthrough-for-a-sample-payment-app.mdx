# Build a Sample Payment App

In this walkthrough, you will build a complete, functional Stellar payment application using Nextellar. By the end of this guide, you will have an app that connects to a user's wallet, displays their balance, and allows them to send XLM to another account.

## Prerequisites

- Node.js v20.18.0 or later
- A Stellar wallet browser extension (like Freighter) installed
- Some testnet XLM in your wallet (you can fund it via the [Stellar Laboratory](https://laboratory.stellar.org/#account-creator?network=test))

## Step 1: Scaffold the Project

First, create a new Nextellar project. Open your terminal and run:

```bash
npx nextellar payment-app --typescript --network testnet
```

Navigate into the project directory and start the development server:

```bash
cd payment-app
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser. You should see the default Nextellar starter page.

## Step 2: Set Up Wallet Connection

Nextellar provides a pre-built `<WalletConnectButton />` component and a `useStellarWallet` hook. Let's create our main layout.

Open `src/app/page.tsx` and replace its contents with:

```tsx
'use client';

import { useStellarWallet } from '@/hooks/useStellarWallet';
import { WalletConnectButton } from '@/components/WalletConnectButton';
import PaymentForm from '@/components/PaymentForm';

export default function Home() {
  const { isConnected, address } = useStellarWallet();

  return (
    <main className="min-h-screen p-8 max-w-2xl mx-auto">
      <header className="flex justify-between items-center mb-12">
        <h1 className="text-3xl font-bold">Stellar Pay</h1>
        <WalletConnectButton />
      </header>

      {isConnected ? (
        <div>
          <p className="mb-8 p-4 bg-secondary rounded-lg">
            Connected: <span className="font-mono">{address}</span>
          </p>
          <PaymentForm />
        </div>
      ) : (
        <div className="text-center p-12 border-2 border-dashed rounded-xl">
          <h2 className="text-xl mb-4">
            Please connect your wallet to continue
          </h2>
          <p className="text-muted-foreground">
            You need to connect a Stellar wallet to send payments.
          </p>
        </div>
      )}
    </main>
  );
}
```

## Step 3: Build the Payment Form

Now, let's create the `PaymentForm` component. This component will use the `useStellarPayment` hook provided by Nextellar to handle the transaction.

Create a new file `src/components/PaymentForm.tsx` and add the following code:

```tsx
'use client';

import { useState } from 'react';
import { useStellarPayment } from '@/hooks/useStellarPayment';
import { useStellarBalances } from '@/hooks/useStellarBalances';

export default function PaymentForm() {
  const [destination, setDestination] = useState('');
  const [amount, setAmount] = useState('');

  const { sendPayment, isPending, error, isSuccess } = useStellarPayment();
  const { balances } = useStellarBalances();

  const xlmBalance =
    balances?.find((b) => b.asset_type === 'native')?.balance || '0';

  const handleSubmit = async (e: React.FormEvent) => {
    e.preventDefault();
    if (!destination || !amount) return;

    await sendPayment({
      destination,
      amount,
      assetCode: 'XLM',
    });
  };

  return (
    <form
      onSubmit={handleSubmit}
      className="space-y-6 bg-card p-6 rounded-xl border shadow-sm"
    >
      <div className="flex justify-between items-end mb-4">
        <h3 className="text-lg font-semibold">Send Payment</h3>
        <span className="text-sm text-muted-foreground">
          Balance: {xlmBalance} XLM
        </span>
      </div>

      <div>
        <label htmlFor="destination" className="block text-sm font-medium mb-2">
          Destination Address
        </label>
        <input
          id="destination"
          type="text"
          value={destination}
          onChange={(e) => setDestination(e.target.value)}
          placeholder="G..."
          className="w-full p-3 border rounded-md font-mono text-sm"
          required
        />
      </div>

      <div>
        <label htmlFor="amount" className="block text-sm font-medium mb-2">
          Amount (XLM)
        </label>
        <input
          id="amount"
          type="number"
          step="0.0000001"
          value={amount}
          onChange={(e) => setAmount(e.target.value)}
          placeholder="0.0"
          className="w-full p-3 border rounded-md"
          required
        />
      </div>

      {error && (
        <div className="p-3 text-red-600 bg-red-50 rounded-md text-sm">
          {error.message}
        </div>
      )}

      {isSuccess && (
        <div className="p-3 text-green-600 bg-green-50 rounded-md text-sm">
          Payment sent successfully!
        </div>
      )}

      <button
        type="submit"
        disabled={isPending || !destination || !amount}
        className="w-full bg-primary text-primary-foreground p-3 rounded-md font-medium disabled:opacity-50"
      >
        {isPending ? 'Sending...' : 'Send XLM'}
      </button>
    </form>
  );
}
```

## Step 4: Test the Application

1. Make sure your local development server is running (`npm run dev`).
2. Ensure your Freighter wallet is set to **Testnet**.
3. Click "Connect Wallet" and approve the connection request in Freighter.
4. Your address and XLM balance should now be displayed.
5. Enter a destination address (you can generate a new one in the Stellar Laboratory) and an amount.
6. Click "Send XLM".
7. Your wallet will prompt you to sign the transaction. Approve it.
8. The UI will show a success message once the transaction is confirmed on the Stellar network.

## Conclusion

Congratulations! You've just built a fully functional Stellar payment application using Nextellar. The hooks handled the complex parts: connecting to the wallet, fetching balances, building the transaction, requesting the signature, and submitting it to the network.

Next, try exploring the `useTransactionHistory` hook to display the user's past payments!
