# Write a Guide on Deploying the Docs Site

## Draft Scope

Create a contributor-facing guide that explains how this documentation site is built and deployed.

## Proposed Route

- Published route: `/docs/guides/deployment`
- Source file: `docs/guides/deployment.mdx`

## Required Coverage

- Deployment flow from branch to hosted site
- Required build steps
- Environment requirements
- Link and content validation expectations

## Notes
