# Build an Interactive API Explorer Page

## Draft Scope

Create a documentation-only API explorer that lets readers browse Nextellar APIs by workflow.

## Proposed Route

- Published route: `/docs/api/explorer`
- Source file: `docs/api/explorer.mdx`
- Sidebar placement: `SDK & Reference`

## Explorer Model

- Connect a wallet
- Read account data
- Send payments
- Work with Soroban contracts

## Graceful Degradation

Use Markdown tables, links, and native `<details>` sections so the explorer remains readable without client-side JavaScript.
