# End-to-End Tutorial: Build and Deploy a Stellar dApp

This comprehensive tutorial guides you through building a complete Stellar decentralized application (dApp) from scratch. You'll start with installation, build a full-featured payment application, and deploy it to production.

## What You'll Build

A production-ready Stellar payment application with:

- Multi-wallet connection (Freighter, Albedo, Lobstr, xBull, Hana)
- Real-time balance display
- Send and receive XLM payments
- Transaction history tracking
- Responsive UI with Tailwind CSS
- Production deployment

**Estimated time**: 45-60 minutes

## Prerequisites

Before starting, ensure you have:

- **Node.js** 20.18.0 or later ([Download](https://nodejs.org/))
- **npm**, **yarn**, or **pnpm** package manager
- A Stellar wallet (Freighter recommended for testing)
- Basic knowledge of React and TypeScript

> **Note**: For detailed installation requirements, see the [Installation guide](/docs/getting-started/installation).

## Step 1: Install Nextellar and Create Your Project

Start by creating a new Nextellar project. This scaffolds a complete Next.js application with Stellar integration pre-configured.

```bash
npx nextellar my-stellar-dapp
cd my-stellar-dapp
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser. You should see the Nextellar starter page with a wallet connect button.

### What Gets Installed

The CLI automatically installs:

- **Next.js 16** - React framework with App Router
- **React 19** - Latest React with concurrent features
- **TypeScript 5** - Type safety throughout
- **Tailwind CSS 4** - Utility-first styling
- **@stellar/stellar-sdk** (v14.4.3) - Official Stellar SDK
- **@creit.tech/stellar-wallets-kit** - Multi-wallet support
- **8 Production-Ready Hooks** - Pre-built React hooks for Stellar operations

## Step 2: Set Up Your Testnet Wallet

Before building features, set up a testnet wallet for testing.

1. **Install Freighter Wallet**: [Get Freighter](https://www.freighter.app/)
2. **Switch to Testnet**: In Freighter settings, select "Test Network"
3. **Create a Testnet Account**: Use [Stellar Laboratory](https://laboratory.stellar.org/#account-creator) to create an account
4. **Fund with Testnet XLM**: Use the Friendbot to get free test XLM

> **Note**: Testnet is Stellar's practice network. Never use real funds or mainnet addresses during development.

## Step 3: Build the Payment Interface

Replace the default `src/app/page.tsx` with a complete payment interface:

```tsx
'use client';

import { useState } from 'react';
import { useStellarWallet } from '@/hooks/useStellarWallet';
import WalletConnectButton from '@/components/WalletConnectButton';

export default function Home() {
  const { connected, publicKey, balances, sendPayment } = useStellarWallet();
  const [recipient, setRecipient] = useState('');
  const [amount, setAmount] = useState('');
  const [memo, setMemo] = useState('');
  const [loading, setLoading] = useState(false);
  const [txHash, setTxHash] = useState('');
  const [error, setError] = useState('');

  const handleSend = async (e: React.FormEvent) => {
    e.preventDefault();
    if (!sendPayment) return;

    setLoading(true);
    setError('');
    setTxHash('');

    try {
      const result = await sendPayment({
        to: recipient,
        amount: amount,
        asset: 'XLM',
        memo: memo || undefined,
      });

      setTxHash(result.hash);
      setRecipient('');
      setAmount('');
      setMemo('');
    } catch (err) {
      setError((err as Error).message);
    } finally {
      setLoading(false);
    }
  };

  return (
    <div className="min-h-screen p-8 bg-gradient-to-br from-blue-50 to-indigo-100 dark:from-gray-900 dark:to-gray-800">
      <div className="max-w-2xl mx-auto">
        <h1 className="text-4xl font-bold mb-8 text-gray-900 dark:text-white">
          Stellar Payment App
        </h1>

        {/* Wallet Connection */}
        <div className="mb-8">
          <WalletConnectButton />
        </div>

        {/* Account Info */}
        {connected && (
          <div className="bg-white dark:bg-gray-800 border rounded-lg p-6 mb-8 shadow-lg">
            <h2 className="text-xl font-semibold mb-4 text-gray-900 dark:text-white">
              Account Info
            </h2>
            <div className="space-y-2">
              <div>
                <span className="text-sm text-gray-600 dark:text-gray-400">
                  Public Key:
                </span>
                <p className="font-mono text-sm break-all text-gray-900 dark:text-white">
                  {publicKey}
                </p>
              </div>
              <div>
                <span className="text-sm text-gray-600 dark:text-gray-400">
                  Balance:
                </span>
                <p className="text-2xl font-bold text-gray-900 dark:text-white">
                  {balances[0]?.balance || '0'} XLM
                </p>
              </div>
            </div>
          </div>
        )}

        {/* Payment Form */}
        {connected && (
          <div className="bg-white dark:bg-gray-800 border rounded-lg p-6 shadow-lg">
            <h2 className="text-xl font-semibold mb-4 text-gray-900 dark:text-white">
              Send Payment
            </h2>
            <form onSubmit={handleSend} className="space-y-4">
              <div>
                <label className="block text-sm font-medium mb-2 text-gray-700 dark:text-gray-300">
                  Recipient Address
                </label>
                <input
                  type="text"
                  value={recipient}
                  onChange={(e) => setRecipient(e.target.value)}
                  placeholder="GDEF456..."
                  className="w-full px-3 py-2 border rounded-md focus:ring-2 focus:ring-blue-500 dark:bg-gray-700 dark:border-gray-600 dark:text-white"
                  required
                />
              </div>

              <div>
                <label className="block text-sm font-medium mb-2 text-gray-700 dark:text-gray-300">
                  Amount (XLM)
                </label>
                <input
                  type="number"
                  step="0.0000001"
                  value={amount}
                  onChange={(e) => setAmount(e.target.value)}
                  placeholder="10.5"
                  className="w-full px-3 py-2 border rounded-md focus:ring-2 focus:ring-blue-500 dark:bg-gray-700 dark:border-gray-600 dark:text-white"
                  required
                />
              </div>

              <div>
                <label className="block text-sm font-medium mb-2 text-gray-700 dark:text-gray-300">
                  Memo (optional)
                </label>
                <input
                  type="text"
                  value={memo}
                  onChange={(e) => setMemo(e.target.value)}
                  placeholder="Payment for services"
                  maxLength={28}
                  className="w-full px-3 py-2 border rounded-md focus:ring-2 focus:ring-blue-500 dark:bg-gray-700 dark:border-gray-600 dark:text-white"
                />
                <p className="text-xs text-gray-500 mt-1">Max 28 characters</p>
              </div>

              <button
                type="submit"
                disabled={loading}
                className="w-full px-4 py-2 bg-blue-600 text-white rounded-md font-medium hover:bg-blue-700 disabled:opacity-50 transition-colors"
              >
                {loading ? 'Sending...' : 'Send Payment'}
              </button>
            </form>

            {/* Success Message */}
            {txHash && (
              <div className="mt-4 p-4 bg-green-50 dark:bg-green-900/20 border border-green-200 dark:border-green-800 rounded-md">
                <p className="text-sm font-medium text-green-800 dark:text-green-200 mb-1">
                  Payment Successful!
                </p>
                <a
                  href={`https://stellar.expert/explorer/testnet/tx/${txHash}`}
                  target="_blank"
                  rel="noopener noreferrer"
                  className="text-sm text-green-600 dark:text-green-400 hover:underline break-all"
                >
                  View on Stellar Expert ↗
                </a>
              </div>
            )}

            {/* Error Message */}
            {error && (
              <div className="mt-4 p-4 bg-red-50 dark:bg-red-900/20 border border-red-200 dark:border-red-800 rounded-md">
                <p className="text-sm text-red-800 dark:text-red-200">
                  {error}
                </p>
              </div>
            )}
          </div>
        )}

        {!connected && (
          <div className="bg-white dark:bg-gray-800 border rounded-lg p-8 text-center shadow-lg">
            <p className="text-gray-600 dark:text-gray-300">
              Connect your wallet to send payments
            </p>
          </div>
        )}
      </div>
    </div>
  );
}
```

> **Learn more about the useStellarWallet hook** in the [complete hook documentation](/docs/hooks/use-stellar-wallet).

## Step 4: Add Transaction History

Create a component to display transaction history. Create `src/components/TransactionHistory.tsx`:

```tsx
'use client';

import { useTransactionHistory } from '@/hooks/useTransactionHistory';
import { useStellarWallet } from '@/hooks/useStellarWallet';

export default function TransactionHistory() {
  const { publicKey } = useStellarWallet();
  const { transactions, loading, error } = useTransactionHistory(publicKey);

  if (!publicKey) return null;
  if (loading)
    return <div className="text-center py-4">Loading transactions...</div>;
  if (error)
    return (
      <div className="text-center py-4 text-red-600">
        Error loading transactions
      </div>
    );

  return (
    <div className="bg-white dark:bg-gray-800 border rounded-lg p-6 shadow-lg mt-8">
      <h2 className="text-xl font-semibold mb-4 text-gray-900 dark:text-white">
        Recent Transactions
      </h2>
      <div className="space-y-3">
        {transactions.length === 0 ? (
          <p className="text-gray-600 dark:text-gray-400 text-center py-4">
            No transactions yet
          </p>
        ) : (
          transactions.slice(0, 5).map((tx) => (
            <div
              key={tx.id}
              className="flex justify-between items-center p-3 bg-gray-50 dark:bg-gray-700 rounded-md"
            >
              <div>
                <p className="font-mono text-sm text-gray-900 dark:text-white">
                  {tx.id.slice(0, 12)}...
                </p>
                <p className="text-xs text-gray-600 dark:text-gray-400">
                  {new Date(tx.created_at).toLocaleString()}
                </p>
              </div>
              <a
                href={`https://stellar.expert/explorer/testnet/tx/${tx.id}`}
                target="_blank"
                rel="noopener noreferrer"
                className="text-sm text-blue-600 dark:text-blue-400 hover:underline"
              >
                View ↗
              </a>
            </div>
          ))
        )}
      </div>
    </div>
  );
}
```

Add it to your `page.tsx`:

```tsx
import TransactionHistory from '@/components/TransactionHistory';

// ... in your component JSX, after the payment form:
{
  connected && (
    <div className="mt-8">
      <TransactionHistory />
    </div>
  );
}
```

## Step 5: Test Your Application

Now test your application end-to-end:

1. **Connect your wallet**: Click "Connect Wallet" and approve in Freighter
2. **Verify balance**: Check that your XLM balance displays correctly
3. **Send a test payment**:
   - Enter a recipient address (you can use your own address for testing)
   - Enter an amount (e.g., `1`)
   - Add an optional memo
   - Click "Send Payment"
   - Approve the transaction in your wallet
4. **Verify transaction**: Check the success message and click the Stellar Expert link
5. **Check transaction history**: Verify the transaction appears in the history list

### Common Issues

**Wallet connection fails**:

- Ensure Freighter is installed and unlocked
- Check you're on testnet in Freighter settings
- Try refreshing the page

**Payment fails**:

- Verify recipient address is valid (starts with 'G', 56 characters)
- Ensure you have sufficient XLM balance (minimum 1 XLM reserve)
- Check that amount is greater than 0

**Transaction not appearing**:

- Wait 5-10 seconds for network confirmation
- Check Stellar Expert for transaction status
- Verify you're on testnet, not mainnet

> **For more troubleshooting**, see the [FAQ](/docs/getting-started/faq).

## Step 6: Customize the UI

Nextellar uses Tailwind CSS for styling. Customize the appearance by modifying the Tailwind classes in your components.

### Change the Color Scheme

Update `tailwind.config.ts` to match your brand:

```typescript
import type { Config } from 'tailwindcss';

const config: Config = {
  content: [
    './src/pages/**/*.{js,ts,jsx,tsx,mdx}',
    './src/components/**/*.{js,ts,jsx,tsx,mdx}',
    './src/app/**/*.{js,ts,jsx,tsx,mdx}',
  ],
  theme: {
    extend: {
      colors: {
        primary: {
          DEFAULT: '#your-brand-color',
          foreground: '#your-text-color',
        },
      },
    },
  },
  plugins: [],
};
export default config;
```

## Step 7: Add Advanced Features

Once you have the basics working, explore advanced features:

### Custom Assets

Send custom assets instead of XLM:

```tsx
await sendPayment({
  to: recipient,
  amount: amount,
  asset: {
    code: 'USDC',
    issuer: 'GA5ZSEJYB37JRC5AVCIA5MOP4RHTM335X2KGX3IHOJAPP5RE34K4KZVN',
  },
  memo: memo,
});
```

> **Learn about trustlines** in the [useTrustlines documentation](/docs/hooks/use-trustlines).

### Smart Contracts

Interact with Soroban smart contracts:

```tsx
import { useSorobanContract } from '@/hooks/useSorobanContract';

const { invoke } = useSorobanContract(contractAddress);

const result = await invoke({
  method: 'increment',
  args: [],
});
```

> **See the Soroban contract guide** in the [useSorobanContract documentation](/docs/hooks/use-soroban-contract).

### DEX Integration

Query order books and trade on the Stellar DEX:

```tsx
import { useOfferBook } from '@/hooks/useOfferBook';

const { offers, loading } = useOfferBook(sellingAsset, buyingAsset);
```

> **Learn more** in the [useOfferBook documentation](/docs/hooks/use-offer-book).

## Step 9: Deploy to Production

Deploy your application to a hosting platform. Vercel is recommended for Next.js applications.
