# Add a Complete Reference of All Custom MDX Components

## Draft Scope

Expand the existing MDX custom components guide into a single reference page that lists every globally registered component.

## Proposed Route

- Published route: `/docs/guides/mdx-custom-components`
- Source file: `docs/guides/mdx-custom-components.mdx`

## Source of Truth

Use `src/components/mdx-components.tsx` to identify every registered component and each component source file for prop details.

## Reference Requirements

- List each component.
- Document props.
- Provide a usage example.
- Note constraints for rendering and composition.
