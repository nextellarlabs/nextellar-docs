# Glossary

This glossary is a lightweight starting point for common terms you will see across the Nextellar docs. Definitions stay short on purpose so you can scan them quickly while reading guides, hook references, and integration pages.

---

## Common Terms

### Account

A Stellar identity that can hold assets, sign transactions, and interact with the network.

### Asset

A token on Stellar, such as native XLM or a custom asset issued by an account.

### Asset Issuer

The Stellar account that creates and backs a custom asset on the network.

### Balance

The amount of XLM or another asset held by a Stellar account.

### dApp

A decentralized application that combines a user interface with blockchain-backed actions or data.

### Freighter

A popular browser wallet used to connect Stellar accounts and sign transactions.

### Friendbot

A testnet service that creates and funds accounts with free test XLM.

### Horizon

The Stellar API service used to read account data, balances, transactions, and network activity.

### Mainnet

The live Stellar network where real assets and production transactions are used.

### Memo

A short note attached to a Stellar transaction, often used to identify a payment.

### Network Passphrase

A value that identifies which Stellar network a transaction belongs to.

### Public Key

A public Stellar account address that usually starts with `G`.

### Secret Key

A private signing key that usually starts with `S` and must be kept secure.

### Soroban

Stellar's smart contract platform for building and running on-chain application logic.

### Soroban RPC

The API service used to read Soroban data and submit smart contract transactions.

### Stellar

The blockchain network that Nextellar targets for payments, assets, and wallet-based applications.

### Stellar SDK

The official JavaScript library used to build, sign, and submit Stellar transactions.

### Testnet

The Stellar testing network used for development, experiments, and pre-production verification.

### Transaction

A signed request that changes network state, such as sending a payment.

### Trustline

A Stellar account setting that allows an account to hold a specific issued asset.

### Wallet

A tool such as Freighter or xBull that lets users connect an account and sign Stellar transactions.

### XDR

The encoded transaction format that wallets and Stellar services exchange.

### XLM

The native asset of the Stellar network, often used for balances, fees, and payments.
