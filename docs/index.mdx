# Welcome to Nextellar Documentation

**Nextellar** is a one-step CLI toolkit that bootstraps production-ready Next.js + TypeScript applications with built-in Stellar blockchain support. Get from idea to working dApp in under 2 minutes.

## Quick Start

```bash
npx nextellar my-stellar-app
cd my-stellar-app
npm run dev
```

That's it! You now have a fully functional Stellar dApp with wallet connection, payment hooks, and smart contract integration.

Nextellar eliminates weeks of boilerplate setup. Instead of configuring Stellar SDK, wallet adapters, transaction builders, and error handling from scratch, you get:

- **Instant Setup**: One command to scaffold a complete dApp

- **Best Practices**: Opinionated architecture following industry standards

## Example: Payment Flow

Here's how easy it is to send a payment with Nextellar:

```tsx
import { useStellarWallet } from '@/hooks/useStellarWallet';

export default function SendPayment() {
  const { connected, sendPayment } = useStellarWallet();

  const handleSend = async () => {
    try {
      const result = await sendPayment({
        to: 'GDEF456...',
        amount: '10.5',
        asset: 'XLM',
        memo: 'Payment for services',
      });

      console.log('Success!', result.hash);
    } catch (error) {
      console.error('Payment failed:', error);
    }
  };

  return (
    <button onClick={handleSend} disabled={!connected}>
      Send 10.5 XLM
    </button>
  );
}
```

That's it! No manual transaction building, no XDR encoding, no wallet adapter configuration - it just works.
