# Installation

Get started with Nextellar in under 2 minutes. No global installation required!

## Prerequisites

Before you begin, ensure you have:

- **Node.js** 20.18.0 or later ([Download](https://nodejs.org/))
- **npm**, **yarn**, or **pnpm** package manager
- **Git** for version control (optional but recommended)

### Check Your Node Version

````bash
node --version
# Should output v20.18.0 or higher
```

## Quick Install

The fastest way to create a Nextellar project is with `npx` (no global installation needed):

```bash
npx nextellar my-stellar-app
```

This command will:

1. Scaffold a complete Next.js 16 application
2. Install all dependencies automatically
3. Configure Stellar SDK and wallet adapters
4. Set up TypeScript and Tailwind CSS
5. Create 8 production-ready React hooks

### Alternative: Global Installation

If you plan to create multiple projects, you can install Nextellar globally:

```bash
npm install -g nextellar
nextellar my-stellar-app
```

## CLI Options

Customize your project during scaffolding:

### Basic Usage

````bash
npx nextellar <project-name> [options]
```bash

### Available Options

| Option                   | Description                  | Default                |
| ------------------------ | ---------------------------- | ---------------------- |
| `--typescript`           | Generate TypeScript project  | `true`                 |
| `--skip-install`         | Skip dependency installation | `false`                |
| `--package-manager <pm>` | Choose npm, yarn, or pnpm    | Auto-detected          |
| `--horizon-url <url>`    | Custom Horizon endpoint      | Testnet                |
| `--soroban-url <url>`    | Custom Soroban RPC endpoint  | Testnet                |
| `--wallets <list>`       | Supported wallet adapters    | Auto-detects installed |
| `--install-timeout <ms>` | Installation timeout         | 1200000 (20 min)       |

### Examples

**Create with custom package manager:**

```bash
npx nextellar my-app --package-manager pnpm
````

**Skip installation (install later):**

```bash
npx nextellar my-app --skip-install
cd my-app
npm install
```

**Configure for mainnet:**

```bash
npx nextellar my-app --horizon-url https://horizon.stellar.org
```

**Select specific wallets:**

```bash
npx nextellar my-app --wallets freighter,xbull
```

## What Gets Installed?

Your new Nextellar project includes:

### Core Dependencies

- **Next.js 16** - React framework with App Router
- **React 19** - Latest React with concurrent features
- **TypeScript 5** - Type safety throughout
- **Tailwind CSS 4** - Utility-first styling

### Stellar Integration

- **@stellar/stellar-sdk** (v14.4.3) - Official Stellar SDK
- **@creit.tech/stellar-wallets-kit** - Multi-wallet support
- **Zustand** - Lightweight state management

- **Lucide React** - Beautiful icon library

- **ESLint + Prettier** - Code quality tools

## Running Your Project

After installation completes:

````bash
cd my-stellar-app
npm run dev
```bash

Open [http://localhost:3000](http://localhost:3000) in your browser. You should see the Nextellar starter page with a wallet connect button!

## Verify Installation

Test that everything works:

1. **Development server starts**: `npm run dev` should run without errors
2. **TypeScript compiles**: No type errors in the console
3. **Wallet connection works**: Click "Connect Wallet" button
4. **Hot reload works**: Edit `src/app/page.tsx` and see changes instantly

Need help? Jump to [Troubleshooting](#troubleshooting).

## Troubleshooting

### Installation fails during scaffolding

If `npx nextellar my-stellar-app` stops while installing dependencies, retry manually from the generated folder:

```bash
cd my-stellar-app
npm install
```bash

Fixes:

- Slow network or registry timeouts: `npx nextellar my-stellar-app --install-timeout 3600000`
- npm registry issues: `npm config set registry https://registry.npmjs.org/`
- Disk or quota issues: free at least 500MB before retrying

### Unsupported Node version

If the scaffold fails before install, confirm your Node version:

```bash
node --version
```bash

Fixes:

- Update to Node.js 20.18.0 or later.
- Use a node version manager like `nvm` or `fnm`.

### Port 3000 already in use

If `npm run dev` fails because the port is occupied:

```bash
npm run dev -- -p 3001
```bash

Fix:

- Start the app on a different port with `-p <port>`.

### TypeScript or build errors after install

If `npm run build` reports TypeScript or type-check errors:

```bash
npm run build
```bash

Fixes:

- Confirm dependencies installed correctly: `rm -rf node_modules package-lock.json && npm install`
- Re-run the scaffold command with the same package manager used in the project

### Module not found or dependency resolution issues

If a dependency is missing after installation:

```bash
rm -rf node_modules package-lock.json
npm install
```bash

Fixes:

- Clear the package cache and reinstall dependencies
- Confirm the project was created with your expected package manager
